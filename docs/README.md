# 미션 - 개발자 비상근무

## 기능 목록
1. 비상 근무 월, 시작 요일 입력
    - [x] 월이 숫자인지 확인
    - [x] 
2. 평일 순번 닉네임 입력
    - [x] 5자 이하인지 확인
    - [x] 중복되는 이름이 있는지 확인
    - [x] 최소 5명의 근무자가 있는지 확인
    - [x] 최대 35명인지 확인
3. 휴일 순번 닉네임 입력
   - [x] 5자 이하인지 확인
   - [x] 중복되는 이름이 있는지 확인
   - [x] 최소 5명의 근무자가 있는지 확인
   - [x] 최대 35명인지 확인
4. 비상 근무 리스트 생성
